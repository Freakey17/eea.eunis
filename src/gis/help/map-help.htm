EUNIS Map help

The Eunis GIS tool is designed to provide geographical information about Eunis sites, like finding a site, identifying a site, zooming and panning around the Europe map.

This tool need the Macromedia Flash Player® to be installed, this player is free and can be found here:.

Map engine information

The map engine is based on EEA ArcIMS services, provided by geonode.eea.europa.eu. The service name is eunis_europe. You can access this service using Eunis GIS tool application, or using a different application, like ArcExplorer. In the same time, you can access this service using any GIS software supporting the OpenGIS® protocols WMS 1.1.1 and WFS 1.0(see www.opengeospatial.org).

Navigation

The Eunis GIS tool can provide navigation over the map.

Zoom

Zoom in: Can be done using drag window (show) or single click (show)

Zoom out. Can be done using drag window (show) or single click as the previous zoom (show).

Zoom all. The map will be displayed at the full extent (show).

Zoom to specific country. The map can be displayed at the extent on a selected country (show). By selecting NONE from the combo box, the highlight over the country will be not displayed

Precise zoom to an site from the selection list. On big and crowded selections, you can zoom to a specific selected site. (show)

Pan/Recenter

Dragging pan. Click an drag the map in the specific direction (show).

You can use CTRL and numeric pad for panning, CTRL+ for zoom in CTRL- for zoom out

Selections

Selecting (Identifying).

Click on the specified point to display information about that specific Eunis site (show).

You can click drag over the map for selection more than one Eunis site (show) , using rectangle and circle selection

You can select Eunis point sites, or Eunis area sites (show).

You can use CTRL and numeric pad 5 to identify everything is visible on screen.

Find site by name

You can search for a site by name or by an part of the name (show).

Clean selection

Any existing selection can be cleared on any moment (show)\

Select by filtering database

You ca select sites just from one database. (show)

Layer management

Each layer can be turned on and off with using the layer list (show).

Layer Information

Eunis Points:

Eunis sites points, V2.2 (June 2005), the layer was derived from Eunis database. The points with no coordinates or with 0,0 coordinates were remove. The sites were converted to a point shapefile.

Eunis Areas

Eunis sites areas, V0.1 (Mai 2005). The layer is derived from Multiple polygon shapes provided by member states. A polygon topology was created.
